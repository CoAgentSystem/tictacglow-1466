# tictacglow-1466

TicTacGlow is a modern, visually stunning Tic-Tac-Toe game designed for quick, engaging gameplay with a premium dark theme and smooth animations. It offers a responsive interface with glassmorphism effects, allowing users to play against an AI or a friend locally, track scores, and enjoy a sleek, interactive experience that enhances the classic game with modern web design.

## Tech Stack
Vanilla JavaScript, HTML5, CSS3

## Setup
Run `docker-compose up` to start the infrastructure.
